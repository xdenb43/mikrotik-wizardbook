# MikroTik Field Guide

> A practical engineering handbook for MikroTik RouterOS.

Welcome to **MikroTik WizardBook** — a curated collection of real-world RouterOS knowledge, architecture decisions, deployment guides, troubleshooting notes, and production-ready configuration examples.

This project is built from practical experience rather than theoretical documentation. Every article is written to explain not only **how** to configure RouterOS, but also **why** a particular solution was chosen, where it should be used, and what limitations should be considered.

---

## What is this project?

Unlike the official MikroTik documentation, this WizardBook focuses on practical engineering scenarios:

- Production-ready RouterOS configurations
- Design and architecture recommendations
- Security hardening
- Performance optimization
- Deployment best practices
- Troubleshooting techniques
- Reusable RouterOS scripts
- Hardware selection guidelines

Every configuration example is intended to be understandable, maintainable, and suitable for real deployments.

---

## Intended audience

This documentation is intended for:

- Network Engineers
- System Administrators
- Home Lab enthusiasts
- Small Business deployments
- Anyone learning MikroTik RouterOS

Whether you are configuring your first MikroTik router or maintaining dozens of devices, the goal is to provide practical guidance instead of isolated configuration snippets.

---

## Documentation principles

Every article follows several simple principles:

- Explain **why**, not only **how**
- Prefer production-ready configurations
- Keep examples simple and reproducible
- Document limitations and side effects
- Avoid unnecessary complexity
- Follow RouterOS best practices whenever possible

---

## Project goals

This project aims to become a long-term engineering reference for MikroTik RouterOS by providing documentation that is:

- Practical
- Accurate
- Easy to navigate
- Easy to reproduce
- Continuously maintained
